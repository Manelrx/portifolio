## Overview

**RiftScore** is a competitive gaming performance analysis platform focused on the **League of Legends** ecosystem. The project was born from the need to centralize player performance metrics in an intuitive and visually appealing environment.

The system processes data from external APIs (Riot Games API), calculates weekly rankings based on performance, and presents detailed statistics in interactive dashboards.

## Technology Stack

| Layer | Technology |
|---|---|
| **Frontend** | Next.js 14, React, TypeScript, Tailwind CSS |
| **Backend** | Next.js API Routes, Server Actions |
| **Data** | Riot Games API, local cache |
| **Hosting** | Cloudflare Pages |
| **Domain** | riftscore.com.br |

## Key Features

- **Automated weekly ranking** — score calculation based on match metrics
- **Player profiles** — individual statistics, history, and trends
- **Interactive dashboard** — data visualization with charts and indicators
- **Cache system** — API request optimization for the Riot API
- **Responsive design** — optimized experience for desktop and mobile

## Technical Challenges

- Integration with the Riot Games API respecting rate limits
- Processing and normalizing data from multiple matches
- Ranking calculation with custom scoring algorithm
- Deployment and performance optimization on edge computing

## Result

Project conceived, developed, and published **100% independently** — from idea to deployment. Demonstrates the ability to work with complex external APIs, data processing, and final product delivery in production.
